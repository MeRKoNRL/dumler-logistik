# Dumler Logistik

Система управления логистикой и транспортом для компании **Dumler Logistik**.

## Основные функции
- Дневной план поездок
- Назначение водителей и автомобилей
- Отчёты и экспорт
- История поездок
- Панель администратора и ролевая система
- Заявки на отпуск, отгулы, больничные

## Развёртывание

1. Клонируйте репозиторий:
```bash
git clone https://github.com/MeRKoNRL/dumler-logistik.git
cd dumler-logistik
```

2. Установите зависимости:
```bash
npm install
```

3. Запуск локально:
```bash
npm run dev
```

## CI/CD

Проект автоматически деплоится на [Vercel](https://vercel.com/) при пуше в ветку `main`.
